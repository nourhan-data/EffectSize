# EffectSize
Stores the code used for generating effect size values in "Beyond Significance: Promoting Effect Size Measures in Comparing Metaheuristic Algorithms",
Available at: https://doi.org/10.3390/a19030191

The file PermutationTest.m in this project is adapted from:
Laurens R Krol (2026). Permutation Test
Available at: https://github.com/lrkrol/permutationTest/blob/master/permutationTest.m

The file swtest.m was adapted from:
Ahmed BenSaïda (2026). Shapiro-Wilk and Shapiro-Francia normality tests. 
Available at: https://www.mathworks.com/matlabcentral/fileexchange/13964-shapiro-wilk-and-shapiro-francia-normality-tests
